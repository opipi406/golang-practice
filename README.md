# Go言語練習用リポジトリ


# Installation

```bash
git clone https://github.com/opipi406/golang-practice
cd golang-practice
```

```bash
docker network create golang_practice_network
docker-compose -d up
```

# Usage

```bash
docker-compose exec app sh
go run main.go
```
